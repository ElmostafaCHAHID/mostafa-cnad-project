Mostafa Library Management API

Cloud-Native Microservices Architecture using Docker & Nginx API Gateway

Base URL

http://localhost:8080

Book Service

GET /books
Retrieve all books
GET /books/:id
Retrieve single book
POST /books
Body: { "title": "Book Title", "author": "Author Name" }

Member Service

GET /members
Retrieve all members
GET /members/:id
Retrieve single member
POST /members
Body: { "name": "Member Name", "email": "email@example.com" }

Loan Service

GET /loans
Retrieve all loans
POST /loans
Body: { "bookId": 1, "memberId": 1 }

Health Endpoints

/gateway/health
/books/health
/members/health
/loans/health
